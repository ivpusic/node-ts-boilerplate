# ts-boilerplate

Minimal boilerplate for Node.js typescript projects

Included:

- Typescript
- Node.js
- Koa
- Nodemon
- Koa Router
- Supertest

Cloning:

```bash
git clone git@github.com:ivpusic/node-ts-boilerplate.git
```

## Usage

Start server (port 3000):

```bash
npm start
```

Run tests:

```bash
npm test
```
